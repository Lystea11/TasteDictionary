How meanings are chosen

Corpus data : Japanese Web 2011 (jaTenTen11)

The dictionary focuses on meanings that frequently appear in native-speaker corpus data from Sketch Engines. For each word, we searched examples with the concordance tool on Sketch Engine and collected about 200 examples. We then categorized each into definitions. We then picked easy Japanese examples for Japanese learners, around JLPT N4/N5, for each definition.

For idioms: We try to use idioms that are relatively easy for beginners to understand, and complex ones are annotated.
